How It Works

1. CSV Format: First column must be "Firebase ID" (the unique identifier for each person)

2. Update Fields: Other columns can be any field name (Name, Role, Level, Tribe, Squad, Status, etc.)

3. Existing Fields: If a field exists in Firebase, it will be updated with the new value

4. New Fields: If a field doesn't exist, it will be created automatically

5. Empty Values: Empty cells are ignored (won't overwrite existing data)

Example CSV Format:

Firebase ID,Name,Role,Level,Tribe,Squad,Status
abc123def456,Alice Johnson,Eng,IC4,Engineering,Backend,Hired
xyz789ghi012,Bob Smith,PM,M2,Product,Mobile,Hired
